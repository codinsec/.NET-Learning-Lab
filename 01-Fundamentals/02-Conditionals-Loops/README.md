### Loops
Understand different loop types and when to use each:
- **for**: When you know the number of iterations
- **foreach**: Iterating over collections
- **while**: When condition is checked before execution
- **do-while**: When you need at least one execution

### Control Flow
Master `break` to exit loops early and `continue` to skip iterations.
