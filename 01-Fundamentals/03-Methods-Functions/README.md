### Method Basics
Understand how to declare methods, pass parameters, and return values.

### Parameter Types
Learn the different ways to pass data to methods:
- **Value parameters**: Default behavior, copies the value
- **ref parameters**: Pass by reference, can modify original
- **out parameters**: Must assign a value, used for returning multiple values
- **params**: Variable number of arguments

### Method Overloading
Create multiple methods with the same name but different signatures.

### Advanced Features
Explore optional parameters, local functions, and expression-bodied methods.
