### Classes
Classes are blueprints for creating objects. They define the structure and behavior of objects.

### Objects
Objects are instances of classes. Each object has its own state (data) and can perform actions (methods).

### Encapsulation
Learn how to encapsulate data and methods within classes using access modifiers.

### Constructors
Understand how to initialize objects using constructors with different parameters.
